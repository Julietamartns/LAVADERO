# Sistema de Gestión de Lavadero

Aplicación web para la gestión diaria de ingresos, egresos y recaudación de un lavadero de vehículos.

## Características
- Registro de vehículos (Patente, Categoría, Importe).
- Control de turnos (Mañana/Tarde).
- Historial de registros por calendario.
- Configuración de precios por categoría.
- Gestión de personal (Roles: Jefe/Empleado).
- Almacenamiento local (LocalStorage).

## Tecnologías
- HTML5 / CSS3 / JavaScript
- Tailwind CSS (CDN)
- FontAwesome (Iconos)

## Instalación
Simplemente clona el repositorio y abre `index.html` en tu navegador.
